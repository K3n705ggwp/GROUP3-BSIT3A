OBJECTIVES:

General Objective:

The study aimed to develop a Parcel Lockbox with Centralized Management System for the purpose of providing a secure, automated, and efficient solution for unattended deliveries, while addressing the challenges of missed deliveries and enhancing last-mile deliveries.

Specific Objectives:

Specifically, this study aimed to:

  1. Design and develop a smart parcel lockbox system that can:
    1. secure parcels using PIN-based access mechanism;
    2. facilitate Cash-on-Delivery (COD) transaction through an automated money dispenser;
    3. capture and verify parcel deliveries using an ESP32 camera and object detection via Edge Impulse;
    4. enable landlords, business managers, or school administrators to rent out compartments for secure deliveries via a centralized management system.
  2. Evaluate the system using ISO 25010â€™s eight (8) software quality standards namely: Functional Suitability, Performance Efficiency, Compatibility, Usability, Reliability, Security, Maintainability, and Portability.
